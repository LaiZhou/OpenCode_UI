## 1.1.0

### New Features

- **Custom Base Path Support** (PR #16): Added a "Custom base path" dropdown in the connection dialog, allowing users to specify a working directory for OpenCode terminal sessions. The dropdown is automatically populated with all detected project modules. The selected path is persisted across sessions.

### Fixes

- **CLI Install Command** (PR #17): Fixed the OpenCode CLI install command in README to match the official download instructions.
