## 1.0.5 Release Notes

### 新特性 (New Features)
- **Diff 进度提示**: Diff Viewer 标题栏现在显示当前审查进度 `(Index of Total)`，例如 `app.kt (1 of 8)`，提升多文件审查体验。
- **连接配置持久化**: 自动记忆上次连接的 Server 地址、Web 模式偏好和密码（使用 Base64 简单加密存储），减少重复输入。
- **Web 模式剪贴板支持**: 内置 JS Polyfill，修复了内嵌浏览器中无法使用 `navigator.clipboard` 导致的报错问题。
- **Web 模式鉴权**: 实现了基于 RequestHandler 的底层 Header 注入，确保 Web 界面能无缝通过 Basic Auth 认证。

### 优化 (Improvements)
- **VFS 写入同步**: 重构了 Accept/Reject 的文件写入逻辑，使用 IntelliJ VFS API (`VfsUtil.saveText`) 替代 Java IO，确保 IDE 能实时感知文件变化、触发索引更新并支持 Undo 操作。
- **Diff 自动推进**: 优化了 Accept/Reject 后的窗口切换逻辑，操作后自动加载下一个待审查文件。
