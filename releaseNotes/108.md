## 1.0.8

### 新功能

- **CLI 自动检测**: 插件现在会自动检测 `~/.opencode/bin/opencode` 路径下的 OpenCode CLI，即使该路径未添加到系统 PATH 环境变量中。

### 修复与优化

- **Linux/Mac 兼容性**: 修复了因 IDE 未继承 Shell PATH 环境变量而导致的 "OpenCode CLI not found" 错误。用户不再需要手动创建 `/usr/local/bin/opencode` 软链接。
- **Windows 连接修复**: 修复了 Windows 平台下因 OpenCode 服务启动较慢导致的连接超时问题，该问题曾导致无法将选中的代码发送到终端。
- **增强连接稳定性**:
  - 将服务启动等待时间延长至 30 秒。
  - 即使初始连接超时，也会在后台持续尝试重连。
  - 增加了 HTTP 请求的超时宽限。
