OpenCode UI

OpenCode UI is an open-source AI coding agent integration that brings the power of opencode.ai directly into your JetBrains IDE development workflow.

Key Features

  • Quick Launch: Instantly create or focus an "OpenCode" terminal session with a single shortcut.
  • Context Sharing: Seamlessly send code selections or file references (with line numbers) from your editor or project view directly to the AI agent.
  • Integrated Sidebar: Access the tool quickly via the OpenCode icon in the right sidebar.

Follow these steps to set up and start using the OpenCode plugin:

  1. Install a compatible JetBrains IDE, such as IntelliJ IDEA, WebStorm, PyCharm, or other IntelliJ-based IDEs (version 2025.2 or later).
  2. Install the OpenCode CLI: This plugin acts as a bridge to the CLI tool. Ensure you have it installed on your system by running:
    npm install -g opencode
  3. Launch your IDE, open Settings (Preferences on macOS), and navigate to Plugins.
  4. Search for OpenCode in the Marketplace and click Install.
  5. Once installed, press Cmd + Esc (Mac) or Ctrl + Esc (Windows/Linux) to launch your first session.

Usage and Shortcuts

  • Toggle Terminal: Use Cmd + Esc (Mac) or Ctrl + Esc (Win/Linux) to jump to the OpenCode session.
  • Add Context: Select code or a file and press Opt + Cmd + K (Mac) or Ctrl + Alt + K (Win/Linux) to send the reference to the terminal.

Plugin Compatibility

Complementary Plugins: This plugin requires the Terminal plugin (which is bundled by default in most JetBrains IDEs) to function correctly.

Conflicting Plugins: There are no known conflicting plugins. If you have custom keymaps assigned to the default OpenCode shortcuts, you can easily reconfigure them in Settings | Keymap.
